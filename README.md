# 요구사항

### 근태현황
- 금일 연차자 인원수가 보여진다.
- 평균 근무시간이 보여진다.
- 신규 입사자가 보여진다. 
- 재택근무자 인원을 보여진다

### 이번 주 실적왕
 - 이번 주 실적이 높은 5개의 부서를 리스트로 뽑아서 보여줍니다.
 
### 출근 관리
- 시간별 출근자 수를 테이블 형식으로 나타낸다.

### 급여 관리
- 급여 조회 :
    - 부서,직급 별 평균 급여를 조회할 수 있어야 합니다.
